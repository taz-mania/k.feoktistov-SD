Проектирование *Сервиса запросов информации* о наличии товаров на полках в режиме реального времени (*API* определения наличия товаров на полках) для региональной ритейловой компании "eDA".

*Название*: On-shelf Availability API : Real-Time Store Inventory Query Service.

Ссервис дает быстрые ответы "есть ли товар на полке" и "в каком магазине рядом" для клиентов и сотрудников магазинов или сервисов доставки с мобильными устройствами.

*Цель создания сервиса*: уменьшить отмены заказов и повысить конверсию.

Сервис предполагает высокую нагрузку:

* потоковые обновления остатков из PoS и Cистемы управления складом (WMS),
* конфликтующие источники истины,
* обработку пиковых значений запросов на наиболее популярные SKU.
