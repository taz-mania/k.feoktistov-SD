## 1. Требования

### 1.1. Функциональные требования

1. Сервис должен предоставлять возможность клиенту через мобильное приложение проверять наличие товара на полках конкретного магазина с привязкой к торговому залу, исключая складские запасы.
2. Сервис должен обеспечивать клиенту функцию геопоиска ближайших магазинов в заданном радиусе, где товар физически присутствует на полке и не зарезервирован под другие заказы.
3. Система должна предоставлять сотрудникам торгового зала через handheld-терминалы возможность сканирования SKU для подтверждения фактического наличия на полке и оперативной коррекции остатков при обнаружении пересорта.
4. Сервис должен реализовывать механизм резервирования товара непосредственно на полке при оформлении клиентом заказов на самовывоз (Click&Collect) или экспресс-доставку.
5. Система должна поддерживать функцию подписки клиента на push-уведомления о поступлении конкретного товара на полку выбранного магазина.

### 1.2. Требования интеграции (API & Интеграции)

1. Сервис должен предоставлять API (REST/gRPC) для синхронного запроса наличия товара по SKU и идентификатору магазина.
2. Сервис должен поддерживать batch-запросы для проверки нескольких позиций SKU одновременно.
3. Система должна обеспечивать потоковую интеграцию с кассовыми терминалами (POS) для приема событий продаж, возвратов и аннулирований чеков в реальном времени.
4. Сервис должен интегрироваться с системой управления складом (WMS) для учета перемещений товаров со склада на торговую точку, а также данных проведенных инвентаризаций.
5. Система должна реализовывать алгоритм разрешения конфликтов между источниками данных: POS, WMS, ручные корректировки, приоритизируя POS-данные, как наиболее актуальные отражение фактических продаж.
6. Сервис должен обеспечивать управление распределенным кэшем с настраиваемыми параметрами и механизмом проактивной инвалидации при получении событий изменения остатков.
7. Система должна поддерживать пространственный индекс для выполнения геопоиска магазинов с фильтрацией по фактическому наличию товара на полке.
