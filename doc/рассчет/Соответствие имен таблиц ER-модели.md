Вот таблица соответствия названий таблиц:

| Английское название | Русское название (в ER-модели) | Описание сущности |
|:---|:---|:---|
| **INVENTORY_EVENTS** | **ЖУРНАЛ_СОБЫТИЙ** | Append-only журнал фактов изменения остатков (Event Store) |
| **RESERVATION_EVENTS** | **ЖУРНАЛ_БРОНИРОВАНИЙ** | Поток событий жизненного цикла резервирования |
| **CATEGORY** | **КАТЕГОРИИ_ТОВАРОВ** | Иерархический справочник категорий номенклатуры |
| **SKU** | **ТОВАРЫ** | Справочник SKU (номенклатура) |
| **STORE** | **МАГАЗИНЫ** | Справочник торговых точек |
| **ASSORTMENT** | **АССОРТИМЕНТ** | Матрица наличия товара в магазине |
| **GEO_LOCATION** | **ГЕОЛОКАЦИИ** | Координаты и адресные детали локаций |
| **USER** | **ПОЛЬЗОВАТЕЛИ** | Учетные записи сотрудников и администраторов |
| **USER_ROLES** | **РОЛИ_ДОСТУПА** | Справочник ролей системы |
| **USER_ROLE_ASSIGNMENTS** | **НАЗНАЧЕНИЕ_РОЛЕЙ** | Связующая таблица пользователей и ролей |
| **CACHE_CONFIG** | **НАСТРОЙКИ_КЭША** | Конфигурация политик кэширования (автономная) |
| **RESERVATION** | **РЕЗЕРВЫ** | Заказы на бронирование товаров |
| **RESERVATION_STATUS** | **СТАТУСЫ_РЕЗЕРВОВ** | Справочник статусов бронирования |
| **RESERVATION_ITEMS** | **ПОЗИЦИИ_РЕЗЕРВА** | Строки состава резервирования (товары) |
| **USER_RESERVATION_LIMITS** | **ЛИМИТЫ_ПОЛЬЗОВАТЕЛЕЙ** | Антифрод-ограничения на количество активных резервов |
| **RESERVATION_OUTBOX** | **ИСХОДЯЩИЕ_СОБЫТИЯ** | Outbox для гарантированной публикации в шину (Saga) |
| **INVENTORY_SNAPSHOT** | **СРЕЗЫ_ОСТАТКОВ** | Денормализованные агрегаты для чтения (Read Model) |
| **TIME_SERIES_POINTS** | **ВРЕМЕННЫЕ_РЯДЫ** | Точки измерений метрик velocity и stock-out |
| **CONFLICT_RECORD** | **КОНФЛИКТЫ_ДАННЫХ** | записи о расхождениях между POS и WMS |
| **RESOLUTION_POLICIES** | **ПРАВИЛА_СВЕРКИ** | Бизнес-политики разрешения конфликтов |
